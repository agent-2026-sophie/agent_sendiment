# tmux 简介
`tmux` （Terminal Multiplexer，终端复用器）是一个终端复用工具，用于在单个终端窗口中访问和管理多个终端会话。允许在一个终端窗口同时开启多个会话（session），每个会话又可以包含多个窗口（window），每个窗口进一步可以切分成多个面板（pane）。适合在远程服务器上工作，即使断开SSH连接，tmux会话中的进程也会在后台继续运行，下次连接时可以无缝恢复。

**核心概念**：

| 概念 | 描述 |
| --- | --- |
| Server | tmux 后台进程，管理所有会话。只要 Server 不杀掉，所有任务都在。 |
| Session | 最高层级的管理单元，代表一个工作上下文（如：数据分析、后端开发）。 |
| Window | 类似于浏览器的“标签页”，一个会话可以有多个窗口。 |
| Pane | 类似于屏幕的“分屏”，一个窗口可以切分成多个面板。 |
| Prefix | 前缀键（默认为 Ctrl + b）。所有快捷键都必须先按前缀键，再按功能键。 |

# 核心功能
- 会话持久化：即使远程 SSH 连接断开，tmux 中的任务依然在后台运行，重新连接后即可一键恢复现场（断线续命）。
- 多任务并行：支持在一个屏幕内通过分屏（Split Pane）同时查看日志、编辑代码、监控系统，无需频繁切换标签页。
- 远程协作：多个用户可以同时附着（Attach）到同一个 tmux 会话，实时看到彼此的操作。

典型使用场景：

| 场景 | 描述 |
| --- | --- |
| 远程跑批 | 在远程服务器上启动模型训练，即使合上电脑，后台仍在运行任务。|
| 复杂监控 | 左边监控系统指标（如 CPU、内存、磁盘等），右边编辑代码。 |
| 环境隔离 | 不同session处理独立的项目，互补干扰。|

# 安装 tmux
```bash
# macOS
brew install tmux

# Linux (Debian/Ubuntu)
apt install tmux
```

# 常用命令
## 会话管理
```bash
# 新建一个未命名的会话
tmux

# 新建一个名为 "mysession" 的会话
tmux new -s mysession

# 列出当前所有 tmux 会话
tmux ls

# 接入（恢复）名为 "mysession" 的会话
tmux attach -t mysession

# 退出并关闭名为 "mysession" 的会话
tmux kill-session -t mysession

# 关闭所有的 tmux 会话
tmux kill-server

```
## 快捷键前缀
在 tmux 内部，大部分操作都需要先按下前缀键（Prefix），默认是 Ctrl + b。按下前缀键后松开，再按后续的指令键。
1. 窗格管理（pane)：
    - Ctrl+b 然后 % ：左右（垂直）分割当前窗格
    - Ctrl+b 然后 " ：上下（水平）分割当前窗格
    - Ctrl+b 然后 方向键 ：在各个窗格之间切换光标
    - Ctrl+b 然后 x ：关闭当前窗格（会有确认提示）
    - Ctrl+b 然后 z ：将当前窗格最大化 / 恢复正常大小
    - Ctrl+b 然后 空格 ：在内置的窗格布局之间切换
2. 窗口管理（window）：
    - Ctrl+b 然后 c ：创建一个新窗口
    - Ctrl+b 然后 n ：切换到下一个窗口 (Next)
    - Ctrl+b 然后 p ：切换到上一个窗口 (Previous)
    - Ctrl+b 然后 0~9 ：切换到指定编号的窗口
    - Ctrl+b 然后 & ：关闭当前窗口
3. 其他：   
    - Ctrl+b 然后 d ：分离 (Detach) 当前会话，让其在后台运行，回到外部普通终端
    - Ctrl+b 然后 ? ：查看所有快捷键帮助

## 快捷键速查表

| 分类 | 操作目的 | 快捷键 | 对应 tmux 命令 |
| :--- | :--- | :--- | :--- |
| **会话 (Session)** | 离开当前会话 | `Ctrl + b, d` | `tmux detach` |
| | 重命名当前会话 | `Ctrl + b, $` | `tmux rename-session` |
| | 会话选择器（列表展示） | `Ctrl + b, s` | `tmux ls` |
| | 删除当前会话 | 无（输入 `exit`） | `tmux kill-session` |
| **窗口 (Window)** | 创建新窗口 | `Ctrl + b, c` | `tmux new-window` |
| | 重命名当前窗口 | `Ctrl + b, ,` | `tmux rename-window` |
| | 切换到下一个/上一个 | `Ctrl + b, n / p` | `select-window -n / -p` |
| | 关闭当前窗口 | `Ctrl + b, &` | `tmux kill-window` |
| | 通过交互式列表切换 | `Ctrl + b, w` | `list-windows` |
| **面板 (Pane)** | 左右分屏（垂直） | `Ctrl + b, %` | `split-window -h` |
| | 上下分屏（水平） | `Ctrl + b, "` | `split-window -v` |
| | 显示面板编号 | `Ctrl + b, q` | `display-panes` |
| | 当前面板全屏切换 | `Ctrl + b, z` | `resize-pane -Z` |
| | 关闭当前面板 | `Ctrl + b, x` | `kill-pane` |
| | 调整大小（微调） | `Ctrl + b, Alt + 方向键` | `resize-pane -U/D/L/R` |

> 操作技巧： 所有的快捷键操作都是先按住 Ctrl 和 b，放开，然后再按下一个功能键。
